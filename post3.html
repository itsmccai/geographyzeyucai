2020 Election By County in PA

This map is created using a process similar to the second map we made. First, the tracts data for Miami were joined with the relevant tracts dataset, and a new field, AREA, was created and calculated based on the geographic area of each tract. From there, three different densities were calculated: Population Density per Tract, Percent Occupied Housing per Tract, and Percent Vacant Housing per Tract. Similarly, for this map, the attribute table of Pennsylvania Counties was joined with the County Voting Table using county names as the common field. A new field, CATEGORY, was then added to represent the supporting rate, and each category was assigned distinct colors for visualization.
